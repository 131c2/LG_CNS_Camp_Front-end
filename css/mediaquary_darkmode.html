CSS 다크 모드 예시

CSS의 prefers-color-scheme 미디어 쿼리를 사용한 다크 모드 전환 예시입니다.

CSS 미디어 쿼리로 다크 모드 구현하기

사용자의 시스템 설정에 따라 자동으로 다크 모드를 감지하는 CSS 코드:

@media (prefers-color-scheme: dark) {
  body {
    background-color: #121212;
    color: #e0e0e0;
  }
}

다크 모드의 장점

  • 어두운 환경에서 눈의 피로 감소
  • OLED 디스플레이에서 배터리 절약
  • 일부 사용자에게는 가독성 향상
  • 현대적인 디자인 트렌드
